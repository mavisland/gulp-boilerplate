# gulp-boilerplate

A boilerplate for building web projects with [Gulp](https://gulpjs.com/) 4.x.

## Quick Start

```
# 1 Clone this repo
git clone https://github.com/mavisland/gulp-boilerplate.git

# 2 Navigate into the repo directory
cd gulp-boilerplate

# 3 Install all node packages
npm install

# 4 Get started
gulp
```
